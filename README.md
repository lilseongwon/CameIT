# 과제 설명
## 내가 먹고 싶은 빵이 가게에 남아있는지 확인할 수 있는 서비스
- 인기 빵의 수량이 적을 때 품절위기 알림을 보내준다
- 가게 오픈/클로즈 시 알림을 보낼 수 있다(어드민)
- 빵 검색과 상세정보 확인이 가능하다
## 사용 스택
- Spring Security
- Spring Data Validation
- JPA / Hibernate
- Firebase(FCM)
- AWS(EC2, Elastic Cash(Redis), RDS(MySQL)
- CI/CD(Github Action)
